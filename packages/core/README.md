# UJL Core

This package contains the core functionality of the UJL Framework. It provides the essential tools needed to process `.ujl` files and enable modular, platform-independent web development.

## Contents
- **Renderer**: Converts `.ujl` files into rendered ContentFrames.
- **Validator**: Ensures the syntax and structure of `.ujl` files are valid.
- **Utilities**: Shared functions and helpers for the core system.

## Note
The contents of this package are under development and will be updated as new features are implemented.
